------------------Usage------------------
To run the program, first import as project into eclipse.

Modify "config.txt" to achieve different configuration preferences.

Run "Process.java" in folder "connect" as java application.

If test on a single machine, every time running a single process, open up another console window, run and then pin it.

Follow the instruction line that appear to enter ID, starting from 0. e.g. If test on 4 processes, then number to be entered should be 0, 1, 2, 3.

Once see "Enter your command." appears in console, you are ready to enter your input.
